# phylocancer
This repository contains analyses and scripts for the phylocancer paper. We list the software requirements, and instructions on how to install and use the software below. A demo and example usage of the software on our datasets is also provided. 

## Requirements
Recommended to use a linux operating system, such as Ubuntu v x.x.x 

GPU hardware and installation of Beagle is recommended to achieve the best computational efficiency. 

Software requirements: 
* BEAST v2.7
* Phylonco v1.2.1
* FLC v1.2.0
* CCD v1.0.0
* Beagle library v4.0.1
* beast-classic v1.6.3
* MASCOT v3.0.7
